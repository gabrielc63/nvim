# NeoVim Distribution

## Copy/Paste easy install (OS X)

```sh
git clone https://github.com/gabrielc63/nvim.git ~/.config/nvim
nvim +PlugInstall +qall
brew install python
brew install python2
pip3 install neovim
pip2 install neovim
brew install ripgrep
```
run :CheckHealth on neovim
